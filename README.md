# Food RPG

**Version:** 0.1.0

Food RPG is an accessible, menu-driven, audio-first RPG designed for the blind community and anyone who prefers text-to-speech, screen readers, keyboard navigation, and accessible output.

## Status

🚧 Under active development.

## Accessibility

Accessibility is a core design goal of Food RPG.

* Audio-first gameplay
* Fully keyboard-driven navigation
* Screen reader friendly
* Designed with NVDA and other text-to-speech software in mind
* Clear, accessible output without reliance on visual elements

## Build Requirements

* Operating System - Windows 10/11
* Screenreader: NVDA 2025 or later
* programming language: Python 3.14 or later
* interface: PyGame
* Speech backend: Prismatoid/PRISM
