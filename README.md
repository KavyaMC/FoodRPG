# Food RPG

**Version:** 0.1.0

Food RPG is an accessible, menu-driven, audio-first RPG designed for the blind community and anyone who prefers text-to-speech, screen readers, keyboard navigation, and accessible output.

## Status

Under active development.

Version 0.1.0 establishes the core application framework and accessibility foundation required for future gameplay development.

## v0.1.0 - Application Framework

This release focuses on building the foundation of Food RPG.

### Implemented Features

- Application startup and lifecycle framework
- Screen architecture framework
- Controller architecture framework
- Main menu framework
- Settings framework
- Save/load persistence foundation
- Accessibility menu framework
- Help documentation framework
- Credits documentation framework
- Notification framework
- New Game framework

### Accessibility

Accessibility is a core design goal of Food RPG.

Implemented accessibility foundations include:

- Fully keyboard-driven navigation
- Screen reader friendly controls and output
- Structured menu navigation
- Speech integration through the PRISM speech backend
- Clear accessible output without reliance on visual elements

## Roadmap

### v0.2.0 - Gameplay Foundation

The next milestone begins gameplay system development.

Gameplay mechanics, progression, economy, and business simulation systems will be introduced progressively through future releases.

## Build Requirements

- Operating System - Windows 10/11
- Screenreader: NVDA 2025 or later
- Programming language: Python 3.14 or later
- Interface: PyGame
- Speech backend: Prismatoid/PRISM
