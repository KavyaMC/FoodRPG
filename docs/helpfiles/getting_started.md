# Getting Started

When Food RPG starts, you are presented with the Main Menu.

Available options include:

- Continue
- New Game
- Settings
- Help
- Credits
- Quit

To start crafting your very first recipes:

1. Select New Game.
2. Complete character creation.
3. Choose a save slot.
4. Begin your journey.

After your character has been created, you can later use Continue to load your saved progress.

During gameplay, the pause menu allows you to save your progress and return to the main menu.
