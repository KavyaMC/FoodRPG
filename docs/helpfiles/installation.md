# Installation

System Requirements

Operating System: Windows 10 or Windows 11

Programming Language: Python 3.14 or later (development builds)

Screen Reader: NVDA 2025 or later is recommended.

Dependencies

PyGame

PRISM (Prismatoid)

Installation

If using a packaged release:

1. Run the installer.
2. Launch Food RPG from the Start Menu or Desktop shortcut.

If using the ZIP release:

1. Extract the archive.
2. Keep the folder structure intact.
3. Launch FoodRPG.exe.

Food RPG stores user settings and save data under:

AppData\Local\FoodRPG

Removing the game does not automatically remove saved data.
